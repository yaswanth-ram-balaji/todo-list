## ✨ Features

- 🧊 Glassmorphism User Interface
- ➕ Add, ✅ Mark, and 🗑️ Delete tasks
- 📊 Live task progress with tracker
- 📱 Responsive layout for all devices
- 🔄 Real-time UI updates
- 🎯 Minimal, clean, and modern design



## 🛠️ Tech Stack

- **HTML5**
- **CSS3**
- **Vanilla JavaScript**
